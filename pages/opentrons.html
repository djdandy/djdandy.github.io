Welcome to the Opentrons' notes page - this page is dedicated to installing the Opentrons application locally, running (and simulating) aliquoting procdeures, and utilizing custom Python procedures alongside minimal custom hardware.

For networked connections with the Opentrons' instrument, one can chose to connect via WiFi (need to know network password/security) or via ethernet connection. The USB board with the ethernet port can be accessed directly if one is willing to DIY.

Useful places provided by Opentrons: Tutorials, Complex Common Commands, Protocol Library, Default Labware, Creating Labware, Default Hardware, and Default Examples.

Similarly, my GitHub page for Opentrons provides plenty of slightly and significantly altered Python procedures for the Opentrons' instrument alongside a custom piece of hardware.
